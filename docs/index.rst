PARAGON Documentation
====================

🚀 Parallel Graph Engine for High-Performance Computing

---

📌 Overview
-----------

PARAGON is a high-performance C++ parallel graph processing engine
supporting algorithms like:

- BFS
- PageRank (push/pull)
- Connected Components
- SSSP
- Triangle Counting
