PARAGON Tutorials
=================

This section discusses tutorials for PARAGON, covering core concepts such as
graph creation, working with weighted graphs, and executing parallel graph
algorithms efficiently.

PARAGON is designed for high performance graph processing, and these tutorials
will guide you step by step from basic usage to advanced parallel execution.

If you are new to PARAGON, we recommend starting with:

- :doc:`../install` Installation across different platforms
